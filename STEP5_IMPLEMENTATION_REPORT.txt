================================================================================
INVARIANT SURFACING IN UI
================================================================================

Equipment Status Values:
- 'complete' - has family, industry, and templates
- 'unmapped' - no family mapping
- 'missing_industry' - has family, no industry
- 'missing_templates' - has family and industry, no templates

Data Structure per Equipment:
{
  id,
  type_name,
  type_code,
  family_code: null | string,  // null = unmapped
  industries: [],              // empty = no industry
  template_count: 0,           // 0 = missing templates
  status: 'complete' | 'unmapped' | 'missing_industry' | 'missing_templates'
}

Gap Resolution Action Items:
- gap_type: which invariant violated
- action_required: human-readable fix description
